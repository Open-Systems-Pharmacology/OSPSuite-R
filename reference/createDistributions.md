# Creates the parameter distributions based on the given individual `individualCharacteristics`

Creates the parameter distributions based on the given individual
`individualCharacteristics`

## Usage

``` r
createDistributions(individualCharacteristics)
```

## Arguments

- individualCharacteristics:

  Characteristics of the individual to create as an instance of
  `OriginData`

## Value

An array of `ParameterValue` containing the value of each individual
parameter
