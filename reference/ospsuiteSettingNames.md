# Names of the settings stored in ospsuiteEnv. Can be used with `getOSPSuiteSetting()`

Names of the settings stored in ospsuiteEnv. Can be used with
[`getOSPSuiteSetting()`](https://www.open-systems-pharmacology.org/OSPSuite-R/reference/getOSPSuiteSetting.md)

## Usage

``` r
ospsuiteSettingNames
```

## Format

An object of class `list` of length 11.
