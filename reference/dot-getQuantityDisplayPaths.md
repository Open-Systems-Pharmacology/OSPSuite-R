# Retrieves the display path of the quantity defined by path in the simulation

Retrieves the display path of the quantity defined by path in the
simulation

## Usage

``` r
.getQuantityDisplayPaths(paths, simulation)
```

## Arguments

- paths:

  A single string or array of paths path relative to the `Simulation`

- simulation:

  A `Simulation` used to find the entities

## Value

a display path for each entry in paths
