# get OSP Dimensions from OSPSuite.Dimensions.xml data

get OSP Dimensions from OSPSuite.Dimensions.xml data

## Usage

``` r
.getOspDimensions(xmlData)
```

## Arguments

- xmlData:

  XML data from
  [`.parseDimensionsXML()`](https://www.open-systems-pharmacology.org/OSPSuite-R/reference/dot-parseDimensionsXML.md)

## Value

a list of supported dimensions
