# Returns a list containing all properties `propertyName` from the .NET objects `netObjects`

Returns a list containing all properties `propertyName` from the .NET
objects `netObjects`

## Usage

``` r
.getPropertyValues(netObjects, propertyName)
```

## Arguments

- netObjects:

  List of .NET object

- propertyName:

  Property name that should be retrieved from the `netObjects`
