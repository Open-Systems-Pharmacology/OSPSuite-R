# Loads a data repository (typically observed data) from a PKML file and returns the loaded observed data. The pkml files are typically exported from PK-Sim or MoBi

Loads a data repository (typically observed data) from a PKML file and
returns the loaded observed data. The pkml files are typically exported
from PK-Sim or MoBi

## Usage

``` r
.loadDataRepositoryFromPKML(filePath)
```

## Arguments

- filePath:

  Full path of pkml file containing the observed data to load
